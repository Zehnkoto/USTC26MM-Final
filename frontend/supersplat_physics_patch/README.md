# SuperSplat Physics Patch

The actual modified SuperSplat files are stored under
`src-overrides/supersplat-src/` so they can be copied back into an upstream
SuperSplat checkout with the same relative paths.

The patch adds the interactive physics workflow: simulation UI, Gaussian
selection, fixed/movable controls, material and solver parameters, backend
submission, and motion playback.

Upstream: https://github.com/playcanvas/supersplat
