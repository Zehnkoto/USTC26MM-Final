# 3DGS / RoMa Reconstruction Tools

This folder contains scripts used to organize RoMa/COLMAP/3DGS comparison runs.
The scripts assume external checkouts of RoMa and the official 3D Gaussian
Splatting implementation.

Included code is limited to orchestration, dataset preparation, result packaging,
and preview helpers. It does not include RoMa weights, 3DGS checkpoints, training
outputs, or third-party source trees.
